充电宝、锂电池类物品安检规定

民用航空安全检查工作手册第一百三十八条

（一）充电宝、锂电池禁止作为行李托运，随身携带时有以下限定条件(电动轮椅使用的锂电池另有规定): 1.标识全面清晰，额定能量小于或等于 100Wh; 2.当额定能量大于 100Wh、小于或等于 160Wh 时必须经航空公司批准且每人限带两块。

（二）可能存在安全缺陷的锂电池，如废弃锂电池、生产厂家召回的锂电池、破损或鼓包的锂电池等。

液态物品携带规定 - 相关规定

国内航班液态物品携带规定

  旅客乘坐国内航班时，液态物品禁止随身携带（航空旅行途中自用的化妆品、牙膏及剃须膏除外）。 航空旅行途中自用的化妆品必须同时满足三个条件（每种限带一件、盛放在单体容器容积不超过100mL的容器内、接受开瓶检查）方可随身携带，牙膏及剃须膏限带一件且不得超过100g（mL）。旅客在同一机场控制区内由国际、地区航班转乘国内航班时，其随身携带入境的免税液态物品必须同时满足三个条件（出示购物凭证、置于已封口且完好无损的透明塑料袋中、经安全检查确认）方可随身携带，如果在转乘国内航班过程中离开机场控制区则必须将随身携带入境的免税液态物品作为行李托运。

携运人体捐献器官 - 相关规定

人体捐献器官携运安检规定

  携运人体捐献器官的，优先安检。应当要求出具移植中心器官接收确认书和公共航空运输企业或其地面服务代理机构书面同意承运证明，核查人体器官获取组织工作人员身份信息与移植中心机关接收确认书信息一致，核验人体器官运输专用标志，对运输箱及内容物通过民航行李安检设备进行检查，无疑点的不再进行开箱检查，并留存移植中心器官接收确认书副本和公共航空运输企业或其地面服务代理机构书面同意承运证明放行。

  保存人体捐献器官所必需的液态物品不受液态物品民航运输条件的限制，人体器官获取组织工作人员及其携运的其他行李物品应当按正常程序接受检查。

  上述物品需要使用危险品作为冷藏（冻）剂的，凭公共航空运输企业或其地面服务代理机构书面同意承运证明放行。

充电宝、锂电池类 - 携带方法

充电宝、锂电池类物品安全携带方法

安全携带充电宝、锂电池的正确方法

  1. 选择合规产品：购买充电宝或锂电池时，应选择正规品牌和渠道，确保产品有清晰的标识，包括额定能量、制造商信息等。避免购买三无产品或改装产品。
  2. 查看额定能量：随身携带的充电宝、锂电池额定能量应小于或等于100Wh。当额定能量大于100Wh、小于或等于160Wh时，必须经航空公司批准且每人限带两块。
  3. 单独存放：携带时，应将充电宝、锂电池单独存放，避免与钥匙、硬币等金属物品混放，以防短路。最好使用原厂包装或绝缘袋进行包装。
  4. 禁止托运：充电宝、锂电池禁止作为行李托运，必须随身携带。
  5. 避免挤压碰撞：携带过程中，应避免充电宝、锂电池受到挤压、碰撞或高温环境，以免造成内部损坏或安全隐患。
  6. 定期检查：使用前应检查充电宝、锂电池是否有破损、鼓包、发热等异常情况，如有异常应停止使用并妥善处理。

雪崩救援背包 - 携带方法

雪崩救援背包安全携带方法

安全携带雪崩救援背包的正确方法

  1. 提前申请批准：携带雪崩救援背包前，应提前向航空公司申请并获得批准，确保符合航空公司的运输规定。
  2. 检查包装状态：确保雪崩救援背包的包装方式能够保证不意外启动，背包内的空气袋必须安装减压阀。
  3. 携带相关文件：携带雪崩救援背包时，应准备好航空公司或其地面服务代理机构的书面同意承运证明，以备安检人员核查。
  4. 配合安检检查：接受安检时，应主动向安检人员说明携带的雪崩救援背包情况，并配合安检人员对背包及内容物进行检查。
  5. 正确使用设备：在携带过程中，应确保雪崩救援背包的触发装置处于安全状态，避免误操作引发安全问题。

海航客舱宠物运输 - 携带方法

海航客舱宠物安全携带方法

安全携带宠物的正确方法

  1. 确认航空公司规定：携带宠物前，应详细了解海南航空公司关于客舱宠物运输的规定，确保宠物符合运输条件。
  2. 准备相关文件：携带动物卫生监督所出具的《动物检疫合格证明》、宠物疫苗注射证明，并填写《客舱运输宠物协议书》。
  3. 选择合适容器：使用符合航空公司要求的宠物运输容器，确保容器坚固、透气、舒适，底部有吸水垫。
  4. 提前训练宠物：携带宠物前，应提前对宠物进行训练，使其适应运输容器，减少在飞行过程中的焦虑和不适。
  5. 提前到达机场：携带宠物乘机时，应提前到达机场，预留足够的时间办理宠物运输手续和接受安检。
  6. 飞行中照顾宠物：飞行过程中，应保持宠物运输容器处于自己视线范围内，并根据需要给予宠物适当的安抚。

液态物品携带 - 携带方法

液态物品安全携带方法

安全携带液态物品的正确方法

  1. 了解携带限制：乘坐国内航班时，液态物品禁止随身携带（航空旅行途中自用的化妆品、牙膏及剃须膏除外）。
  2. 控制容器容积：航空旅行途中自用的化妆品必须盛放在单体容器容积不超过100mL的容器内，每种限带一件，并接受开瓶检查。
  3. 集中存放：将液态物品集中存放在透明的可重新密封的塑料袋中，便于安检人员检查。
  4. 特殊液态物品：对于保存人体捐献器官所必需的液态物品，不受液态物品民航运输条件的限制，但需出具相关证明文件。
  5. 配合安检检查：接受安检时，应主动将液态物品从行李中取出，配合安检人员进行检查。

携运人体捐献器官 - 携带方法

人体捐献器官安全携运方法

安全携运人体捐献器官的正确方法

  1. 提前沟通协调：携运人体捐献器官前，应提前与航空公司、机场安检部门沟通，说明情况并协调优先安检。
  2. 准备完整文件：携带移植中心器官接收确认书和公共航空运输企业或其地面服务代理机构书面同意承运证明。
  3. 使用专用容器：将人体捐献器官存放在专用的运输箱中，确保运输箱符合卫生和安全要求，并带有明显的人体器官运输专用标志。
  4. 保持合适条件：根据器官保存要求，确保运输过程中温度等条件合适，必要时使用冷藏（冻）剂，并携带相关证明文件。
  5. 配合优先安检：接受安检时，主动出示相关文件，配合安检人员对运输箱及内容物通过民航行李安检设备进行检查。
